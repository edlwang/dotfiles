# Quickstart
```bash
git clone https://www.github.com/edlwang/dotfiles.git "$HOME/dotfiles/"
bash "$HOME/dotfiles/init.sh"
```

## Starship
```bash
mkdir -p ~/.local/bin
curl -sS https://starship.rs/install.sh | sh -s -- -b ~/.local/bin
```

## uv
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

## venv
```bash
uv venv py313 --seed
```
